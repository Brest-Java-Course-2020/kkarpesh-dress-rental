## Environment setting
```
install git
install docker
install docker-compose
```

## Installing
Select the directory for the project and clone the project from github:
```
$ git clone https://github.com/kkarpesh/dress-rental.git
```

## Build project
Run command in project directory:
```
$ gradle build
```

## Run project
Run command in project directory:
```
$ docker-compose up
```
